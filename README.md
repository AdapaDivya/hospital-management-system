# 🏥Hospital Management System

A console-based **Hospital Management System** built using **Java (OOP concepts)**.  
This project simulates basic hospital operations like patient management, doctor management, and hospital services using a menu-driven interface.

---

## 🚀 Project Overview

**MEDIASSIT** is a beginner-friendly Java project designed to demonstrate core Object-Oriented Programming concepts by building a real-world hospital management simulation.

It focuses on managing hospital data in a structured and modular way using classes and objects.

---

## ✨ Features

- 👨‍⚕️ Add and manage patient details  
- 🩺 Add and manage doctor records  
- 📋 View hospital data using menu system  
- 🧠 Implements OOP principles:
  - Encapsulation  
  - Classes & Objects  
  - Abstraction (basic level)  
- 💻 Console-based interactive UI  
- 📦 ArrayList used for in-memory storage  

---

## 🛠️ Technologies Used

- Java (JDK 8)
- Object-Oriented Programming (OOP)
- ArrayList (Collections)
- Scanner (User Input)
- Console-based Application

## ▶️ How to Run

### 1. Clone the repository
git clone https://github.com/your-username/hospital-management-system.git

### 2. Navigate to project folder
cd hospital-management-system

### 3. Compile Java files
javac *.java

### 4. Run the application
java HospitalMain
